# ✨ Emojis - 쉽고 빠른 글로벌 이모지 검색 & 복사 서비스

**Emojis**는 인스타그램, 트위터(X), 블로그, 그리고 개발 문서 등 어디서나 필요한 이모지를 가장 빠르고 편리하게 찾아서 복사할 수 있는 웹 앱입니다.

### 🔗 서비스 링크
[https://hounga13.github.io/emojis](https://hounga13.github.io/emojis)

---

### 🌟 주요 특징

*   **⚡ 빛보다 빠른 검색 & 복사**: 검색창에 단어를 입력하고, 원하는 이모지를 클릭하기만 하면 즉시 클립보드에 복사됩니다.
*   **🌍 19개국 다국어 완벽 지원**: 한국어, 영어, 일본어, 중국어, 스페인어 등 전 세계 19개 주요 언어로 UI와 검색 키워드를 지원합니다. 사용자의 브라우저 언어를 자동으로 감지하여 최적의 경험을 제공합니다.
*   **📱 모든 기기 최적화**: PC는 물론 모바일에서도 한눈에 들어오는 반응형 디자인과 부드러운 애니메이션을 경험할 수 있습니다.
*   **🕒 최근 사용한 이모지**: 내가 자주 사용하는 이모지는 상단에 자동으로 정리되어 매번 찾을 필요가 없습니다.
*   **📂 체계적인 카테고리**: 스마일리, 동물, 음식, 여행 등 이모지 표준 카테고리별로 깔끔하게 정리되어 있어 탐색이 즐겁습니다.

### 🚀 사용 방법

1.  **검색**: 상단 검색창에 '웃음', 'apple', '강아지' 등 찾고 싶은 단어를 입력하세요.
2.  **선택**: 결과 목록에서 마음에 드는 이모지를 찾습니다.
3.  **복사**: 이모지를 터치하거나 클릭하세요. 끝! 이제 원하는 곳에 붙여넣기(`Ctrl+V` 또는 `Cmd+V`) 하시면 됩니다.
